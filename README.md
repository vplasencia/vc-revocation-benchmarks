# Merkle Tree-based Revocation Benchmarks

This repository contains benchmarks of Merkle Tree implementations (SMT and LeanIMT) for revocation in verifiable credentials.

## Circuit Benchmarks

To generate circuit benchmarks, go to the [`circuits`](./circuits/) folder and follow the instructions from the README file. 

## Browser Benchmarks

To generate browser benchmarks, go to the [`browser`](./browser/) folder and follow the instructions from the README file. 

## Node.js Benchmarks

To generate Node.js benchmarks, go to the [`node`](./node/) folder and follow the instructions from the README file. 

## Smart Contracts Benchmarks

To generate smart contract benchmarks, go to the [`contracts`](./contracts/) folder and follow the instructions from the README file.
